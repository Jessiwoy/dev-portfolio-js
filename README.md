# Dev Portfolio (HTML, CSS, JS)

Landing page de portfólio pessoal feita com **HTML + CSS + JavaScript (Vanilla)**, sem frameworks.

## Objetivo
- Código organizado e escalável
- Manipulação de DOM com JS
- UI moderna e profissional
- Conteúdo em português

## Como rodar
Abra o `index.html` no navegador ou use a extensão **Live Server** no VSCode.

## Estrutura
- `index.html`
- `assets/css` (reset + estilos)
- `assets/js` (scripts)
- `assets/img` (imagens e favicon)
