Portfolio Landing Page
📌 Sobre o Projeto

Este projeto consiste no desenvolvimento de uma Landing Page de portfólio profissional, criada com o objetivo de apresentar informações pessoais, projetos desenvolvidos e formas de contato de maneira clara, moderna e responsiva.

A aplicação foi construída utilizando HTML, CSS e JavaScript Vanilla, sem o uso de frameworks ou bibliotecas externas. O objetivo foi demonstrar domínio dos fundamentos do desenvolvimento front-end, aplicando boas práticas de organização de código, responsividade, acessibilidade básica e estruturação semântica.

O layout segue uma estética minimalista em tema escuro, inspirada em interfaces modernas utilizadas em portfólios de desenvolvedores e produtos digitais.

🛠 Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando apenas tecnologias fundamentais da web:

HTML5
Estruturação semântica da página e organização do conteúdo.

CSS3
Estilização da interface, layout responsivo e criação de animações leves.

JavaScript Vanilla
Implementação de interações e manipulação básica do DOM.

Não foram utilizados frameworks ou bibliotecas externas, com o objetivo de demonstrar conhecimento sólido das bases do desenvolvimento front-end.

📂 Estrutura do Projeto

A aplicação é composta pelas seguintes seções principais:

Header

Contém a navegação principal do site e um menu adaptado para dispositivos móveis.

Hero Section

Apresentação inicial com informações básicas sobre a desenvolvedora.

About

Seção com uma breve descrição profissional.

Projects

Área dedicada à exibição de projetos selecionados.

Contact

Seção com formas de contato, incluindo e-mail e link para GitHub.

Footer

Rodapé com informações adicionais, links externos e ano atualizado automaticamente via JavaScript.

⚙️ Funcionalidades Implementadas

O projeto inclui diversas funcionalidades voltadas para melhorar a experiência do usuário e demonstrar boas práticas de desenvolvimento:

Layout totalmente responsivo

Menu mobile para navegação em telas menores

Animações de entrada utilizando Intersection Observer

Botão "Back to Top" para facilitar a navegação

Ícones SVG inline para GitHub, e-mail e links externos

Ano dinâmico no footer, atualizado automaticamente com JavaScript

Estrutura preparada para futura implementação de modal de projetos

Também foram aplicadas práticas básicas de SEO, incluindo:

meta description

Open Graph

Twitter Cards

arquivo robots.txt

arquivo sitemap.xml

💻 Como Executar o Projeto Localmente

Como se trata de um projeto estático, não é necessário instalar dependências.

Clone o repositório:

git clone https://github.com/Jessiwoy/dev-portfolio-js.git

Acesse a pasta do projeto:

cd nome-do-repositorio

Abra o arquivo index.html em seu navegador.

Você também pode utilizar extensões como Live Server no VS Code para visualizar o projeto com atualização automática.
